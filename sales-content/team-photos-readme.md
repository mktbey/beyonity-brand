# Team Foto-Konvention

Teamfotos für den Sales Hub — werden unter `assets/sales/team/` abgelegt.

## Anforderungen

| Eigenschaft | Vorgabe |
|-------------|---------|
| Format | JPG |
| Abmessungen | Quadratisch, mindestens 256 × 256 px |
| Dateiname | lowercase, nur ASCII-Zeichen, Umlaute ersetzen |
| Hintergrund | Neutral (weiß, grau oder Büroumgebung) |

## Upload

Foto in `assets/sales/team/` ablegen und committen:

```
git add assets/sales/team/vorname-nachname.jpg
git commit -m "feat: add team photo — vorname nachname"
git push
```
