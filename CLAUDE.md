# Beyonity Brand Context

Full brand guidelines: https://github.com/mktbey/beyonity-brand/blob/main/Brand.md

## Colors
- Primary Blue:    #3c57e1
- Yellow (CTA):    #f8ff76
- Dark Blue:       #001a64
- Light Blue:      #d8ddf9
- White:           #ffffff
- Gray:            #efefef
- Error Red:       #ff7575
- Success Green:   #a7faa9

## Typography
- Font: Bw Gradual (ExtraBold = headlines, Medium = body)
- Fallback: Sora (Google Fonts)
- Line-height: 120% headlines, 140% body

## Buttons
- Normal:  bg #f8ff76, text #001a64, font Sora ExtraBold, fully rounded
- Hover:   + shadow 0 0 6px rgba(0,0,0,0.5)
- Active:  bg #3c57e1, text #ffffff

## Shapes
- Only circles and rounded rectangles — no sharp corners
- Consistent border-radius within a medium
- No shadows except on hover/clickable elements

## Copy Rules
- Language: German (Sie-form), English for headlines ok
- Tone: bold, solution-oriented, warm — tech-driven but not cold
- Always use concrete numbers ("48 Millionen Klicks")
- No: "revolutionär", "synergetisch", "best-in-class" without proof
- CTAs: "Jetzt entdecken", "See in Action", "Demo buchen"
- Never: "Mehr erfahren"

## Logo
- Assets: /assets/logo/
- Only Dark Blue (#001a64) or White on dark backgrounds
- Clear space: height of logo letterform on all sides
